# AI Studio Overview

AI Studio is a full-stack AI workspace designed for cloud and local AI workflows.

It supports Gemini cloud mode for public deployment and Ollama local mode for development and experimentation.

## Core Purpose

AI Studio helps users:

- Chat with AI
- Upload and analyze documents
- Build searchable knowledge bases
- Index websites
- Import social knowledge
- Use OCR for images and scanned content
- Run locally with Ollama or in the cloud with Gemini
