# Installation Guide

## Requirements

- Python 3.12+
- Git
- PostgreSQL or SQLite/local database
- Gemini API key for cloud mode
- Ollama for local mode

## Basic Setup

```powershell
git clone https://github.com/Karim-1211/ai-studio.git
cd ai-studio
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

Configure environment variables based on cloud or local mode.
