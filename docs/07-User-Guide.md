# User Guide

## Start a Chat

1. Login.
2. Create or select a conversation.
3. Ask a question.
4. Optionally enable knowledge search.

## Upload Attachments

1. Click the attachment button.
2. Upload PDF, DOCX, TXT, or image.
3. Ask questions about the uploaded file.

## Use Knowledge

1. Add global knowledge or website knowledge.
2. Enable knowledge in chat.
3. Ask questions based on indexed content.
