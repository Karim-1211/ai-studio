# Developer Guide

## Development Workflow

1. Work locally.
2. Validate with compile commands.
3. Commit changes.
4. Push to GitHub.
5. Let Render deploy.
6. Verify live app.

## Validation

```powershell
python -m compileall -q .
```

## Release Workflow

Each release should include:

- Setup guide
- Validation guide
- Rollback guide
- Changelog
- Release notes
