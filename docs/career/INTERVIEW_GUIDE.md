# AI Studio Interview Guide

## 30-Second Explanation

AI Studio is a full-stack AI workspace that lets users chat with AI, upload documents, index websites, import social knowledge, and ask questions using RAG. It runs in the cloud with Gemini and locally with Ollama.

## Why I Built It

I wanted to create a professional AI application that combines real business use cases with modern AI engineering, including RAG, OCR, authentication, PostgreSQL, Docker, and cloud deployment.

## Technical Talking Points

- Flask backend architecture
- PostgreSQL persistence
- Gemini cloud provider
- Ollama local mode
- RAG document pipeline
- OCR image processing
- Website indexing
- Social knowledge import
- Render deployment
- Environment-based configuration

## Strong Interview Answer

AI Studio taught me how to move from a local prototype to a production-ready AI platform. The most important challenge was separating local Ollama assumptions from cloud Gemini deployment while keeping a single maintainable codebase.
