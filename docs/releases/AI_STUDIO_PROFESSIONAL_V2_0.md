# AI Studio Professional Edition v2.0

Professional Edition v2.0 improves the real-time chat experience.

## Highlights

- ChatGPT-style streaming remains enabled.
- Stop generation now preserves partial output.
- Stopped responses are saved to chat history.
- Provider warning text now supports both cloud and local mode.

## Compatibility

- Gemini cloud mode: supported.
- Ollama local mode: supported.
- Database migration: not required.
- Render environment changes: not required.
